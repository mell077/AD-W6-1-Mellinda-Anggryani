outputnya meow itu karena kode ini menggunakan konsep Polimorfisme dan Method Overriding. var a dengan tipe data Animal, objek asli yang dimasukkan ke dalam Cat.
Saat perintah a.sound() dijalankan, Java tidak melihat label variabelnya, melainkan melihat wujud asli objek di dalamnya. Karena objek aslinya adalah Cat dan saya sudah menulis ulang (menimpa) metode sound() di kelas tersebut, maka Java secara otomatis menjalankan versi milik si kucing. Itulah alasan kenapa output-nya adalah "Meow", karena perilaku spesifik milik kelas anak selalu diprioritaskan saat program berjalan.
